👾 Javascript

Javascript es un lenguaje de programación dinámico y de alto nivel para crear interfaces interacticas y dinámicas.

Es uno de los tres pilares fundamentales del desarrollo web.

  • 1️⃣ HTML: Define la estructura del contenido. Son los elementos que aparecen en la página web.
  • 2️⃣ CSS: Define la apariencia, colores, tamaños, estilos, etc.
  • 3️⃣ JS: Define el comportamiento, es decir lo que hace la página o sus elementos.

Características de Javascript

  1. 👉 Lenguaje Interpretado: No se necesita compilar. El navegador lo entiende directamente.
  2. 👉 Basado en Objetos: Maneja los elementos de html como objetos: window, date, array, class.
  3. 👉 Dinámico: Podemos modificar el contenido en tiempo real. (Sin recargar)
  4. 👉 Vésatil y fácil de aprender: Aunque nacio especialmente para la web, se usa también para servidores, aplicaciones móviles, videojuegos, animación y por supuesto para IA.
